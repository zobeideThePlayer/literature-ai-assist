To Run the Application

  1. Backend Setup:
  cd backend
  python -m venv venv
  source venv/bin/activate  # or `venv\Scripts\activate` on Windows
  pip install -r requirements.txt

  # Create .env file with your DeepSeek API key
  cp .env.example .env
  # Edit .env and add: DEEPSEEK_API_KEY=your_key_here

  # Start backend
  uvicorn app.main:app --reload

  2. Frontend Setup:
  cd frontend
  npm install
  npm run dev

  3. Access the app at: http://localhost:5173

  Features

  - Search PubMed + Semantic Scholar for papers
  - AI analysis with chain-of-thought reasoning
  - Visual timeline showing how insights were derived
  - Generate structured literature review for peer-reviewed articles
  - Export to Markdown
